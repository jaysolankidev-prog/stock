# AHGO Stock Management System

This is a Laravel-based stock management system with features like:
- Inline editing for Bag NWT (Net Weight)
- Category-based grouping (Bags, Bundles, Square Fit)
- Activity logging for all stock updates
- Real-time total calculations

## Getting Started

1. Clone the repository
2. Run `composer install`
3. Run `npm install && npm run dev`
4. Configure your `.env` file
5. Run `php artisan migrate --seed`
6. Run `php artisan serve`
